# 设计师的工作过程

1. 发现并定义问题
2. 通过调查与研究来深入理解问题
3. 创造性地提出构思
4. 精炼构想并时间
5. 进一步迭代

# 收集需求五连问

1. 业务目的：为什么要做这个功能
2. 业务目标：产品期望得到怎样的成果
4. 目标用户：谁来用这个功能
5. 用户需求：用户为什么要这个功能
6. 用户行为：如何让大家都用这个功能
